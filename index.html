Halo, saya adalah seorang junior web programmer yang memiliki kemampuan membuat website dalam waktu singkat tanpa bantuan tim. Saat ini, saya tengah mencari partner untuk berbisnis bersama di berbagai bidang. Hobi saya adalah menemukan solusi untuk masalah-masalah yang saya alami sendiri. Saya percaya bahwa setiap tantangan adalah peluang untuk belajar dan berkembang. Saya juga memiliki antusiasme besar terhadap teknologi dan inovasi.

SMK Muhammadiyah Bligo

Saya adalah seorang pelajar yang pernah mengalami beberapa tantangan di sekolah ini. Saya pernah mengikuti lomba IT networking tingkat kabupaten sebagai pengalaman pertama saya dalam kompetisi tersebut, dan meskipun hasilnya belum memuaskan (peringkat ke-4 dari 4 peserta), saya belajar banyak dari pengalaman tersebut. Selain itu, saya juga pernah diberhentikan dari keanggotaan organisasi IPM di sekolah dengan beberapa perbedaan pendapat. Meskipun demikian, pengalaman-pengalaman ini telah mengajarkan saya banyak hal dan memotivasi saya untuk terus berusaha dan berkembang.
